Un poco sobre mí

Durante 6 años en la industria, descubrí el poder de la tecnología para revolucionar procesos. Hoy, como Técnico Superior en DAW, combino mi conocimiento del mundo físico con mi experiencia en desarrollo web y móvil para digitalizar ideas que resuelven problemas reales.

Experiencia

Desarrollo web

Participé en el desarrollo de una plataforma web completamente funcional y responsive para una empresa externa, destinada al control de visitas y formaciones en distintas sedes. Trabajé en un equipo multidisciplinario utilizando metodologías ágiles (Scrum con sprints), gestionando tareas mediante Trello y utilizando Git/GitHub para el control de versiones. El frontend fue desarrollado con React, Vite y Bootstrap, mientras que el backend y la base de datos se implementaron con Supabase y PostgreSQL.

HTML CSS JavaScript React Supabase

Desarrollo Móvil

Desarrollé de manera independiente una aplicación móvil personalizada para un aficionado al cine, con funcionalidades clave como búsqueda intuitiva de próximos estrenos, películas populares y favoritas. Llevé a cabo todo el ciclo de desarrollo en solitario: desde el análisis de requerimientos hasta el despliegue final, implementando la base de datos Isar, el frontend adaptativo y la integración con APIs externas. Mediante un proceso continuo de feedback con el cliente a través de reuniones periódicas, ajusté la aplicación a sus necesidades específicas, y hasta hoy sigo encargándome de su mantenimiento y mejora continua.

Flutter Dart Isar API

PLC / Scada / HMI

Con 6 años de experiencia en entornos industriales y procesos de producción, he complementado mi formación en desarrollo web y móvil con especialización en programación de PLC, SCADA y HMI. Este conocimiento integrado me permite diseñar soluciones tecnológicas que optimizan líneas de producción, combinando mi experiencia sectorial con mi pasión por la automatización industrial.
